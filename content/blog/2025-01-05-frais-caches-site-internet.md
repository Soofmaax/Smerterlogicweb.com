<p>Un site WordPress à 1 500€ vous coûtera 3 300€ à 5 100€ sur trois ans. Un site statique à 2 490€ vous coûtera 2 490€. Zéro euro de maintenance après. <h2>Le Coût Total de Possession : l’indicateur que personne ne vous montre</h2>
<p>Le TCO, c’est la somme de tous les coûts liés à votre site sur une période donnée : création + hébergement + domaine + maintenance + licences + évolutions. La plupart des prestataires ne vous parlent que du prix de création — comme si on vendait une voiture sans parler de l’essence, de l’assurance et de l’entretien.</p>
<p>Pourtant, c’est sur les frais récurrents que les écarts se creusent. Un site créé pour 800€ peut vous coûter 600€ par an de maintenance : <strong>2 600€ sur 3 ans</strong>. Un site créé pour 2 490€ sans maintenance vous coûte <strong>2 490€ sur 3 ans</strong>. <em>Exigez le TCO</em> avant de signer.</p>

<h2>Les coûts récurrents obligatoires : ce que vous paierez chaque année</h2>
<p><strong>Domaine</strong> : 10–20€/an. Votre adresse web doit être renouvelée annuellement. Certains registrars augmentent le prix au renouvellement (10€ → 25€). Lisez les petites lignes et mettez une alerte calendrier.</p>
<p><strong>Hébergement</strong> : statique = Netlify/Vercel <strong>gratuit</strong>. WordPress mutualisé = 50–120€/an. VPS = 600–4 800€/an (souvent inutile pour une vitrine).</p>
<p><strong>Licences de plugins (WordPress)</strong> : 50–1 200€/an selon fonctionnalités (formulaires, sécurité, sauvegardes, SEO, e‑commerce). Attention aux essais gratuits qui deviennent payants six mois plus tard.</p>
<p><strong>Minimum incompressible/an</strong> : 110–640€ (domaine + hébergement + licences). Si vous avez un WordPress et que vous ne payez rien, c’est que vous n’êtes pas propriétaire ou que le site n’est plus à jour (risque).</p>

<h2>La maintenance technique : le gouffre financier du WordPress</h2>
<p>WordPress = base de données, milliers de plugins, mises à jour fréquentes. Sans maintenance, le site devient une <strong>faille</strong>. 43 % des piratages WordPress viennent de plugins/thèmes obsolètes. Nettoyage : 500–2 000€.</p>
<p>La maintenance inclut mises à jour cœur/plugins/thème, sauvegardes, surveillance, correctifs, support. Tarifs 2025 : <strong>290–1 800€/an</strong> (freelance → agence). Moyenne observée : 396–1 980€/an.</p>
<p>À l’inverse, un <strong>site statique</strong> (HTML/CSS/JS pur) n’a pas de CMS/DB/plugins → <strong>0 maintenance</strong>, vulnérabilité minimale, hébergement gratuit, domaine 10–20€/an.</p>
<p><strong>Sur 3 ans</strong> : WordPress basique 870€ (290€/an) ; WordPress intermédiaire 5 400€ (1 800€/an) ; Statique = 0€ (hors domaine). C’est pour cela qu’un statique à 2 490€ est souvent <em>moins cher</em> sur 3 ans qu’un WordPress à 1 500€.</p>

<h2>Les coûts stratégiques : SEO continu et contenu</h2>
<p>Le SEO est un marathon. Vos concurrents publient et optimisent ; si vous ne faites rien, vous perdez en visibilité.</p>
<p><strong>Accompagnement SEO</strong> : 300–1 500€/mois selon ambition (audit, optimisation, nouvelles pages, suivi concurrents). <strong>Rédaction SEO</strong> : 150–400€/article (1 000–1 500 mots). Ajustez selon la compétitivité de votre secteur.</p>

<h2>Formule Évolution : 20€/mois pour faire évoluer sans rappeler un dev</h2>
<p>Le problème classique : chaque petite modif coûte 80€/h. Résultat : vous ne changez rien, le site vieillit.</p>
<p><strong>Formule Évolution</strong> : 20€/mois, sans engagement, 1h/mois (cumulable 3 mois). Ajout/modif textes, images, sections, petite page, ajustements design, SEO ponctuel, corrections mineures.</p>
<p>Support prioritaire 24 h ouvrées, monitoring (uptime, vitesse), alertes.</p>
<p>Tarifs préférentiels : refonte −30 %, fonctionnalités −25 %, article SEO 99€ (1 000–1 500 mots), nouvelle page avancée −20 %.</p>
<p><strong>Exemple</strong> : 12 mois de petites évolutions = 240€ pour 12 heures (20€/h équivalent) vs 960€ à 80€/h.</p>
<p>Optionnelle : un statique peut tourner des années sans bouger. Idéale si vous voulez un site vivant à budget maîtrisé.</p>

<h2>Modalités de paiement : acompte, échelonné, abonnement</h2>
<p><strong>Acompte</strong> : 50 % à la signature pour sécuriser l’engagement et démarrer (standard). Le prestataire sans acompte ? Méfiance.</p>
<p><strong>Échelonné</strong> : 50 % (signature) + 30 % (maquettes validées) + 20 % (livraison). Exemple Offre Pro 2 490€ : 1 245€ + 747€ + 498€.</p>
<p><strong>Abonnement mensuel</strong> : 120–500€/mois (hébergement, maintenance, petites évolutions). Avantage : zéro prise à tête. Inconvénient majeur : <strong>propriété</strong> — souvent vous ne possédez pas le site ; vous arrêtez, il disparaît. Sur 3 ans : 3 600–18 000€ sans propriété.</p>

<h2>Pièges à éviter : abonnements sournois et clauses cachées</h2>
<ul>
  <li>Cases pré‑cochées (services « VIP », sauvegarde premium). Lisez chaque ligne.</li>
  <li>Renouvellement automatique avec hausse non signalée. Désactivez et mettez une alerte.</li>
  <li>Plugins devenus payants. Demandez la liste des plugins, leur coût annuel, et les alternatives.</li>
  <li>Maintenance avec engagement ≥ 12 mois. Refusez ; préférez résiliation à 1 mois.</li>
</ul>

<h2>FAQ</h2>
<h3>Quels sont les frais annuels minimum pour un site vitrine ?</h3>
<p>WordPress : 340–2 100€/an (domaine 50–300€, hébergement 50–300€, maintenance 290–1 800€). Statique : 10–50€/an (domaine), hébergement gratuit, 0 maintenance.</p>
<h3>La maintenance d’un site WordPress est‑elle vraiment obligatoire ?</h3>
<p>Oui. Sans mises à jour régulières (cœur/plugins/thème), le site devient vulnérable. 43 % des piratages WordPress viennent d’extensions obsolètes. Nettoyage : 500–2 000€. Prévenez : 290–1 800€/an.</p>
<h3>Pourquoi un site statique n’a pas besoin de maintenance ?</h3>
<p>Pas de base de données ni de CMS → moins d’attaque de surface ; code figé, hébergement serveur statique → robustesse et simplicité.</p>
<h3>Qu’est‑ce que la Formule Évolution à 20€/mois ?</h3>
<p>1h/mois cumulable 3 mois ; modifs de contenu/visuel/design mineur, SEO ponctuel, monitoring, support prioritaire, tarifs préférentiels.</p>
<h3>Dois‑je payer un acompte pour la création ?</h3>
<p>Oui : 50 % standard à la signature, puis 30 % à validation des maquettes, 20 % à la livraison. Cela sécurise des deux côtés.</p>
<h3>Abonnement mensuel ou achat comptant ?</h3>
<p>Achat si vous voulez <strong>posséder</strong> et maîtriser vos actifs numériques (statique recommandé). Abonnement si vous acceptez de ne jamais posséder et déléguer totalement — mais attention : à l’arrêt, le site disparaît.</p>
